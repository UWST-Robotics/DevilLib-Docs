# 1 - Hardware/Sensors

## Smart Motors

```cpp
SmartMotor sampleMotor = SmartMotor("Motor", 1);
SmartMotorGroup sampleMotors = SmartMotorGroup("MotorGroupName", {1, -2, 3, -4});
```

## ADI Pneumatics

```cpp
ADIPneumatic pneumatic = ADIPneumatic("Pneumatic", 1);
ADIPneumaticGroup pneumaticGroup = ADIPneumaticGroup("PneumaticGroupName", {1, 2, 3});
```

## Inertial Sensor

```cpp
InertialSensor inertialSensor = InertialSensor("IMUName", 1);
InertialSensorGroup inertialSensorGroup = InertialSensorGroup("IMUGroupName", {1, 2});
```

## Rotation Sensor
```cpp
RotationSensor rotationSensor = RotationSensor("RotationName", 1);
```

## GPS
```cpp
GPS gpsSensor = GPS("GPSName", 1);
```

## ADI Digital IO
```cpp
ADIDigitalInput digitalInput = ADIDigitalInput("DigitalOutName", 1);
```
