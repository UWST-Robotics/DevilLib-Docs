# DevilLib
DevilLib is a PROS library built and maintained by VEX-U team [DevilBots](https://devilbots.org) at the University of Wisconsin-Stout.
It's primarily designed and used internally within DevilBots, but it's open for any other VEX teams to use or expand on.

If you've never programmed a robot before, these guides aim to help teach you everything you need to know about programming in VEX Robotics.
However, it is assumed you have at least a basic understanding of programming in C++.
If this is not the case, [SoloLearn](https://www.sololearn.com/en/learn/courses/c-plus-plus-introduction) is a fantastic resource.
Unlike most things in life, SoloLearn is free**.

***As long as you're not paying for premium like some kind of sicko.*

If you have any questions or run into any issues, feel free to email us at [robotics@uwstout.edu](mailto:robotics@uwstout.edu).

## Features

- 🔧 Flexible [drivetrain](tutorials/drivetrain) support
- 🕹️ Built-in [joystick input curving](advanced/joystickcurve)
- 📝 Automated hardware logging by default
- 🖥️ Customizable [displays](tutorials/displays)

### Autonomous
- 📍 Flexible [odometry](tutorials/odometry) support
- ✏️ Web-based [autonomous maker](https://auto.devilbots.org/)
- 🧮 Trajectory planning w/ [motion profiles](advanced/controlloops/motionprofile)
- ➡️ Trajectory following w/ [RAMSETE](advanced/trajectory/ramsete)
- ➡️ Path following w/ [Basic Pure Pursuit](advanced/paths/basicpurepursuit)
- ⚙️ Built-in feedforward and feedback controllers (including [PID](advanced/controlloops/pid))
